# Train the Upgrade — NETBANKING 2.0

A 5-level browser mini-game where you train a legacy netbanking system through a series of tech-upgrade challenges (mobile-first swings, zero-latency bench press, one-click KYC whack-a-mole, real-time settlement weak-point hunting, and seamless UPI charge & release), culminating in a "NETBANKING 2.0" victory screen.

## Play it

Open `punch-bag-game.html` directly in a browser, or enable GitHub Pages on this repo for a hosted link (see below).

## Live preview via GitHub Pages

Once this repo is pushed to GitHub:

1. Go to the repo on github.com → **Settings** → **Pages**.
2. Under "Build and deployment", set **Source** to **Deploy from a branch**, branch `main`, folder `/ (root)`. Save.
3. GitHub will publish the site at:
   `https://<your-username>.github.io/<repo-name>/punch-bag-game.html`
4. It can take a minute or two for the first deploy to go live.
